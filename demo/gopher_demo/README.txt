Bienvenido al Servidor Gopher de Demostración
=============================================

Este es un servidor Gopher de ejemplo para demostrar el protocolo RFC 1436.

Gopher es un protocolo de Internet distribuido diseñado en 1991 en la 
Universidad de Minnesota. Fue uno de los primeros sistemas de navegación
de documentos en Internet, anterior a la World Wide Web.

CARACTERÍSTICAS:
- Navegación basada en menús
- Protocolo simple basado en texto
- Soporte para diferentes tipos de archivos
- Sin necesidad de HTML o JavaScript

CONTENIDO DE ESTE SERVIDOR:
- Documentos de texto en /docs
- Imágenes en /images  
- Archivos binarios en /files

Para navegar, selecciona cualquier elemento del menú principal.

¡Disfruta explorando este servidor Gopher!
